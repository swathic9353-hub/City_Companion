# City Companion

A full-stack web application that helps users discover and manage city-related services and information.

## Features
- User-friendly interface
- Responsive design
- Real-time data management
- Full-stack architecture

## Technologies Used
- React.js
- Node.js
- Express.js
- PostgreSQL/MySQL
- JavaScript
- HTML/CSS

## Installation

git clone https://github.com/swathic9353-hub/City_Companion.git

cd City_Companion

npm install

npm start
