# Smart Directory App (City Companion)

A full-stack web application designed to help users discover, explore, and manage city-related services and information efficiently. The platform provides a centralized directory for essential city services with an intuitive and responsive user interface.

---

## Project Overview

Smart Directory App simplifies access to city services by allowing users to search and browse various categories such as restaurants, hospitals, hotels, transportation, and other local services. The application provides an organized and user-friendly experience for discovering city resources.

---

## Features

### User Features

* Browse city services and locations
* Search services by category or keyword
* View detailed information about services
* Responsive user interface
* Easy navigation and modern design

### Application Features

* Real-time data management
* Category-based service organization
* RESTful API integration
* Dynamic content rendering

## Installation

### Clone the Repository

```bash
git clone https://github.com/swathic9353-hub/City_Companion.git
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

The application will run locally on:

```text
http://localhost:3000
```
